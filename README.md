# IRead
Dapp for Book Reports.

## Description
This script posts an article selected randomly from Pocket to Slack.

## Requirement
- Node.js
- npm
- Metamask
- Truffle

## Usage

```
git clone https://github.com/theMistletoe/IRead.git
cd IRead
npm i
truffle develop
migrate
cd client
npm i
npm start
```
